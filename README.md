# WOWOROOM 寢飾單頁電商網站
前台 Demo：https://umon752.github.io/JS_WOWOROOM/  
後台 Demo：https://umon752.github.io/JS_WOWOROOM/admin

## 專案說明
**JS 工程師養成直播班 - 2021 春季**  

第九週最終關卡作品  
> 依課程提供的設計稿重新整合視覺

## 使用技術
* 切版：`SCSS` `Bootstrap4` 
* 動態：`Anime.js` `AOS`
* 功能：`JavaScript` `Axios` `C3.js`

## 開發功能
* 前台
  * 取得商品資料
  * 商品類別篩選
  * 加入購物車
  * 修改購物車數量
  * 刪除單筆 / 全部購物車商品
  * 送出預定資料
* 後台
  * 取得訂單資料
  * 刪除單筆 / 全部訂單
  * 圓餅圖
